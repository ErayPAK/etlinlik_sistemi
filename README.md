# Etkinlik Bilet Satış Platformu 🎫

Kullanıcıların güncel etkinlikleri görüntüleyebildiği, detaylarını inceleyebildiği ve dijital bilet satın alabildiği web tabanlı bir platformdur.

## 🚀 Özellikler
* **Etkinlik Listeleme:** Kategorilere göre filtrelenebilir etkinlik havuzu.
* **Bilet Satın Alma:** Hızlı ve güvenli bilet edinim süreci.
* **Kullanıcı Paneli:** Satın alınan biletlerin takibi (varsa ekle).
